Introduction au CSS

Apprends à styliser tes pages web facilement !

Pourquoi utiliser CSS ?

CSS permet de rendre un site web joli et bien organisé.

  • Changer les couleurs
  • Modifier la police et la taille du texte
  • Aligner et positionner les éléments

Comment ça marche ?


        h1 {
            color: red;
        }

Ce code rend tous les titres <h1> rouges.

Les 3 façons d’ajouter du CSS

1. Directement sur un élément


        <p style="color: green;">Je suis vert !</p>

2. Dans un bloc <style> en HTML


        <style>
        p {
            color: green;
        }
        </style>

3. Dans un fichier CSS séparé


        /* styles.css */
        p {
            color: green;
        }

Ajoute ensuite dans ton HTML :


        <link rel="stylesheet" href="styles.css">

Organiser un site avec CSS

1. Centrer un texte


        h1 {
            text-align: center;
        }

2. Ajouter des marges et de l’espace


        p {
            margin: 20px;
            padding: 10px;
        }

3. Changer l’arrière-plan


        body {
            background-color: lightblue;
        }

Animations et effets

Bouton interactif


        button {
            background-color: blue;
            color: white;
            padding: 10px 20px;
            border-radius: 5px;
            transition: background 0.3s;
        }
        button:hover {
            background-color: red;
        }

Le bouton change de couleur au survol !
