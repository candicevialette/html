Cours : Introduction aux Balises HTML

HTML (HyperText Markup Language) est le langage de base pour structurer le contenu d'une page web. Voici un récapitulatif des balises les plus utilisées.

1. Balises de structure

Ces balises définissent la structure de base d'un document HTML.

  • <!DOCTYPE html> : Définit le type de document comme HTML5.
  • <html> : Balise racine de tout document HTML.
  • <head> : Contient les métadonnées du document.
  • <title> : Définit le titre de la page.
  • <body> : Contient le contenu visible de la page.
  • <div> : Crée une section ou un conteneur.

2. Balises de texte

Ces balises sont utilisées pour la mise en forme du texte.

  • <h1> à <h6> : Définit des titres de niveaux différents.
  • <p> : Définit un paragraphe.
  • <strong> : Met un texte en gras (importance).
  • <br> : Insère un saut de ligne.
  • <ul> et <li> : Crée des listes non ordonnées.
  • <img> : Affiche une image avec les attributs src et alt.

3. Balises spéciales

Ces balises ajoutent des fonctionnalités spécifiques.

  • <meta> : Définit les métadonnées du document.
  • <style> : Permet d'intégrer des styles CSS.
  • <header> : Définit l'en-tête d'une page ou section.
  • <section> : Regroupe un contenu logique.

4. Attributs importants

Les attributs modifient le comportement des balises HTML.

  • class : Applique un style CSS spécifique.
  • id : Identifie un élément de manière unique.
  • style : Ajoute du CSS directement dans l'élément.
  • onclick : Déclenche une action au clic.
