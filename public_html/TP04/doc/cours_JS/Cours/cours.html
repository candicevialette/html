Cours Complet sur JavaScript

Apprends à rendre tes pages web interactives avec JavaScript !

Pourquoi utiliser JavaScript ?

JavaScript est utilisé pour ajouter de l’interactivité à un site web.

  • Modifier du contenu en temps réel
  • Créer des animations
  • Gérer des événements utilisateur (clics, saisies, etc.)

Comment fonctionne JavaScript ?

Exemple : Modifier le texte d’un élément en cliquant sur un bouton.

Texte initial


        document.getElementById("changeText").addEventListener("click", function() {
            document.getElementById("text").innerText = "Texte modifié !";
        });

Les 3 façons d’ajouter du JavaScript

1. Directement dans un élément HTML


        <button onclick="alert('Bonjour !')">Clique-moi</button>

2. Dans un bloc <script> en HTML


        <script>
        document.getElementById("text").style.color = "blue";
        </script>

3. Dans un fichier JavaScript séparé


        // script.js
        document.getElementById("changeText").addEventListener("click", function() {
            document.getElementById("text").innerText = "Texte modifié !";
        });

Ajoute ensuite dans ton HTML :


        <script defer src="script.js"></script>

Animations et interactions avec JavaScript

1. Changer la couleur d’un élément

Regarde-moi changer !


        document.getElementById("colorButton").addEventListener("click", function() {
            document.getElementById("colorText").style.color = "green";
        });

2. Afficher ou masquer un élément

Ceci peut être caché.


        document.getElementById("toggleButton").addEventListener("click", function() {
            let text = document.getElementById("toggleText");
            if (text.style.display === "none") {
                text.style.display = "block";
            } else {
                text.style.display = "none";
            }
        });
