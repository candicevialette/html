Fiche de Révision : Balises HTML Utilisées

1. Balises de structure

  • <!DOCTYPE html> : Définit le type de document comme HTML5.
  • <html> : Balise racine de tout document HTML.
  • <head> : Contient les métadonnées du document (titre, encodage, etc.).
  • <title> : Spécifie le titre affiché dans l'onglet du navigateur.
  • <body> : Contient le contenu principal visible de la page.

2. Balises de texte

  • <h1> à <h6> : Définit les titres de différents niveaux.
  • <p> : Définit un paragraphe de texte.
  • <strong> : Met le texte en gras pour indiquer une importance.
  • <u> : Souligne le texte.
  • <mark> : Met en surbrillance le texte.
  • <code> : Indique du texte représentant du code informatique.
  • <blockquote> : Utilisé pour des citations longues.
  • <dd> : Définit une description d'un terme ou d'une donnée.

3. Balises de listes

  • <ul> : Définit une liste non ordonnée (avec des puces).
  • <li> : Définit un élément d'une liste (ordonnée ou non).

4. Balises spéciales

  • <meta> (dans <head>) : Définit les métadonnées du document.
  • <br> : Insère un saut de ligne.

5. Attributs importants

  • class : Permet d'appliquer un style ou de grouper des éléments.
  • id : Identifie de manière unique un élément dans la page.
