TP JavaScript : Exercices

  • Exercice 1 : Calculer une surface
    On rentre les dimensions d’un rectangle et hop, JavaScript fait le calcul et affiche le résultat.
  • Exercice 2 : Valider les valeurs
    Avant d'afficher la réponse, on demande si les valeurs sont bonnes, histoire de ne pas se tromper.
  • Exercice 3 : Vérifier la saisie
    Ici, pas question d’écrire n’importe quoi ! Seuls les chiffres positifs sont acceptés.
  • Exercice 4 : Afficher le résultat sur la page
    Cette fois, fini les alertes, le résultat s'affiche directement sur le site.
  • Exercice 5 : Faire le calcul avec une fonction
    On organise mieux le code en mettant le calcul dans une fonction réutilisable.
  • Exercice 6 : Générer une table de multiplication
    Entre un nombre et regarde sa table de multiplication s'afficher sous forme de tableau.
  • Exercice 7 : Calculer un devis pour du béton
    On doit calculer combien de béton il faut pour une dalle et combien ça va coûter.
  • Exercice 8 : Afficher le devis
    On reprends l'exercice 7 pour qu'un devis apparaisse de la commande.
  • Exercice 9 : Afficher le devis
    Je reprends l'exercice 8 sauf que je remplace l'entreprise de béton par une de couture.
