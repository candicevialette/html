TP JavaScript : Exercices

  • Exercice 1 : Calcul de la surface d'un rectangle - L'utilisateur entre les dimensions via une boîte de dialogue et le résultat s'affiche en alert et console.
  • Exercice 2 : Confirmation des valeurs - L'utilisateur doit confirmer les dimensions avant de voir le résultat.
  • Exercice 3 : Validation de saisie - Seules les valeurs numériques positives sont acceptées.
  • Exercice 4 : Affichage dans la page - Le résultat est affiché directement sur la page web.
  • Exercice 5 : Fonction de calcul - Le calcul est effectué par une fonction JavaScript.
  • Exercice 6 : Table de multiplication - L'utilisateur entre un nombre et voit sa table sous forme de tableau HTML.
  • Exercice 7 : Devis cimenterie - Calcul du volume de béton, du nombre de camions et du coût du devis en fonction des dimensions saisies.
