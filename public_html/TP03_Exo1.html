Fiche de Révision : Balises HTML Utilisées

1. Balises de structure

  • <!DOCTYPE html> : Définit le type de document comme HTML5.
  • <html> : Balise racine de tout document HTML.
  • <head> : Contient les métadonnées du document (titre, encodage, etc.).
  • <title> : Spécifie le titre affiché dans l'onglet du navigateur.
  • <body> : Contient le contenu principal visible de la page.
  • <div> : Définit une division ou une section dans le document HTML.

2. Balises de texte

  • <h1> à <h6> : Définit les titres de différents niveaux.
  • <p> : Définit un paragraphe de texte.
  • <strong> : Met le texte en gras pour indiquer une importance.
  • <br> : Insère un saut de ligne.
  • <ul> et <li> : Utilisées ensemble pour des listes non ordonnées.
  • <img> : Permet d’afficher une image. Attributs importants : src pour le chemin de l’image et alt pour le texte alternatif.
  • <button> : Crée un bouton interactif.

3. Balises spéciales

  • <meta> : Définit les métadonnées du document.
  • <style> : Permet d’intégrer des styles CSS dans le document HTML.
  • <header> : Définit l’en-tête d’une page ou d’une section.
  • <section> : Regroupe une section logique d’un document.

4. Attributs importants

  • class : Permet d'appliquer un style ou de grouper des éléments.
  • id : Identifie de manière unique un élément dans la page.
  • style : Ajoute des styles CSS directement à un élément.
  • onclick : Déclenche une action lorsqu’un utilisateur clique sur l’élément.
